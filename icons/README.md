# 图标文件说明

本目录应包含以下图标文件：

- `icon16.png` - 16x16 像素
- `icon48.png` - 48x48 像素  
- `icon128.png` - 128x128 像素

## 快速创建图标

### 方法一：在线生成

1. 访问 [Favicon.io](https://favicon.io/favicon-generator/)
2. 设计你的图标（文字、表情符号或上传图片）
3. 下载生成的图标包
4. 将对应尺寸的文件重命名并放入此目录

### 方法二：使用设计工具

使用 Photoshop、Figma、Canva 等工具创建：
- 画布尺寸：128x128（创建最大尺寸，然后缩放）
- 格式：PNG（支持透明背景）
- 建议：使用简单、清晰的设计，在小尺寸下也能辨识

### 方法三：使用表情符号（快速方案）

```bash
# 在 macOS 上使用 emoji 快速创建占位符图标
# 或者从网上下载 emoji 图片作为临时图标
```

## 设计建议

- **主题**：视频、相机、批处理相关的图标
- **颜色**：使用品牌色或与界面配色一致
- **风格**：简洁、现代、扁平化设计
- **对比度**：确保在浅色和深色背景下都清晰可见

## 临时解决方案

如果暂时没有图标，可以：
1. 下载任意 PNG 图片
2. 使用在线工具调整为 16x16、48x48、128x128 三个尺寸
3. 保存到此目录
4. 扩展仍可正常使用，只是图标可能不够美观
